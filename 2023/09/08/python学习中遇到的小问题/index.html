问题汇总

2023.09.08 pandas时间相减后的值转换为年为单位

  1. 确保时间1和时间2都为日期型函数 可以使用pd.to_datetime()进行转换
  2. 通过/np.timedelta64(1,’Y’)将两个日期相减后的结果转换为年为单位
  3. 同理可转为天’D’,月等等
  4. 最后.astype(‘int’)将结果转换为整数
